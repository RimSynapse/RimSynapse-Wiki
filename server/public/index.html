AI Agent Tool REST API Guide

Exposing HTTP endpoints allows non-mcp clients and standard scripts to interact directly with local developer tools. Here is how your AI agent can query and execute commands:

GET http://localhost:3000/api/tools
Retrieves the full list of registered tools, inputs, and schemas.
POST http://localhost:3000/api/tools/:name
Executes a specific tool command. Replace :name with the tool name.
Example Javascript Request Payload:
fetch("http://localhost:3000/api/tools/launch_rimworld", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ arguments: { quicktest: true, nosound: true } }) }) .then(res => res.json()) .then(data => console.log(data));
